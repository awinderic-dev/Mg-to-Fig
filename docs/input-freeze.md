# 输入冻结清单（v1）

## 已冻结策略

- 交付：MasterGo 插件 + Figma 插件 + `.mgfig` 迁移资源包
- 资源策略：结构 JSON 只描述节点与资源引用，图片/icon/遮罩/SVG 等真实资源进入包内 `assets/`
- 资源实现口径：插件链路优先 `.mgfig/assets/` + `manifest.json`，不依赖会话态临时 fetch；外链仅作为兜底
- 字体策略：按名称相近度映射（同名 > 同 family/style > 相近名 > 系统回退）
- 验收口径：参数不可变化，仅允许文本换行差异

## 待用户提供

- 三份样例文件链接（简单/复杂/含组件实例）
- 每个样例对应黄金页面链接或页面名

## 默认阈值

- 可接受：文本换行差异
- 不可接受：颜色、字号、字重、行高、圆角、描边、阴影、透明度变更
- 不可接受：图片/icon 因单 JSON 导出而静默丢失；必须有资源文件、可编辑矢量或明确诊断降级
