# 输入冻结清单（v1）

## 已冻结策略

- 交付：MasterGo 插件 + Figma 插件
- 资源策略：小图内嵌，大图外链/分包
- 资源实现口径：插件链路优先 `assets[]`（data/uri），不依赖会话态临时 fetch
- 字体策略：按名称相近度映射（同名 > 同 family/style > 相近名 > 系统回退）
- 验收口径：参数不可变化，仅允许文本换行差异

## 待用户提供

- 三份样例文件链接（简单/复杂/含组件实例）
- 每个样例对应黄金页面链接或页面名

## 默认阈值

- 可接受：文本换行差异
- 不可接受：颜色、字号、字重、行高、圆角、描边、阴影、透明度变更
